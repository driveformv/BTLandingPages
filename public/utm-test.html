What are UTM Parameters?

UTM parameters are tags added to URLs to track the effectiveness of online marketing campaigns across traffic sources and publishing media.

Parameter Description Example
utm_source Identifies which site sent the traffic (e.g., google, newsletter) facebook
utm_medium Identifies what type of link was used (e.g., cpc, email) social
utm_campaign Identifies a specific product promotion or strategic campaign spring_sale
utm_term Identifies search terms (optional) hiring+jobs
utm_content Identifies what specifically was clicked (optional) logo_link

How It Works

When you visit the recruitment landing page with UTM parameters:

  1. The application captures the UTM parameters from the URL
  2. The parameters are stored in the browser's localStorage
  3. When a user submits an application form, the UTM parameters are attached to the submission
  4. This allows you to track which marketing campaigns are generating the most applications
